Insurance records

How to Document Artwork for Insurance Conversations

Documentation helps make an artwork record easier to review. It does not determine coverage, set value, or replace advice from an insurer, broker, appraiser, attorney, or tax professional.

Archivale Collector Notes · 6 minute read

Insurance conversations are easier when each artwork has a clear, dated record. That record should collect what you know, what you can confirm, and which documents support the file. It should also avoid turning private notes into claims about authenticity, attribution, market value, or coverage.

In this note, "insurance-ready documentation" means organized records a collector can review or share when needed: photos, user-confirmed details, supporting documents, user-provided values if tracked, and the date the record was prepared. It does not mean insurer approval, an appraisal conclusion, or a coverage decision.

Create one record for each artwork

Start by giving every artwork its own record instead of keeping the details scattered across texts, email, camera rolls, and folders. Include confirmed identifying details such as artist or maker if known, title or description, date or approximate date, medium, dimensions, edition information if relevant, and current location.

Add photos that help someone understand the object without handling it: front, back, signature or mark, frame, labels, and any visible condition details you want to remember. If you track an insurance value for your own records, label it as user-provided and keep the source or context beside it.

Attach supporting documents

A useful record keeps supporting material close to the artwork. Attach or reference receipts, invoices, appraisal documents if they exist, certificates, provenance notes, repair records, framing invoices, shipping records, condition notes, and relevant correspondence summaries.

Describe each document by what it is and where it came from. A receipt can support acquisition history. A prior appraisal can provide context from that document. A certificate or provenance note can be relevant supporting material. None of those documents should be described by the app or record as automatic proof of authenticity, ownership, attribution, or current value.

Keep value separate from authenticity

Values, authorship, condition, and ownership questions belong in separate lanes. If you enter a value because it is useful for your own records, make the label plain: user-provided insurance value, purchase price, or value from a named document. Do not relabel a private entry as an appraised value unless you are describing an actual appraisal document.

The same discipline helps with attribution. Use wording such as "artist listed on receipt," "family note says," or "needs review" when that is what the source supports. Careful labels make the record easier to discuss without overstating what the archive knows.

Export or share records carefully

When you prepare a report or export, include the report date so the reader knows when the record was assembled. Label user-confirmed fields, attached documents, and user-provided values clearly. If a detail is uncertain, keep that uncertainty visible instead of smoothing it into a stronger claim.

Avoid language that suggests an outside review unless one has actually happened outside the app. A dated export can be useful for a conversation, but it is not an insurer-approved report, a coverage promise, a valuation opinion, or a professional authentication.

Know when to ask a human professional

Ask the relevant professional when the question moves beyond organizing records. Coverage questions belong with an insurer or broker. Appraisal needs belong with a qualified appraiser. Disputed attribution, ownership, legal, tax, or condition concerns need appropriate human review.

Archivale can help keep the record tidy so those conversations start with better context. It should not be treated as the party making the insurance, legal, tax, appraisal, or authenticity decision.

Build the record first

Build one complete record per artwork, then keep receipts, photos, and notes together so the record is easier to review when needed. Keep professional decisions separate from the archive.
